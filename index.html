Tags

O que é uma tag HTML? HTML é uma markup language, o que significa que ela é escrita com códigos que uma pessoa pode ler sem que seja necessário compilá-la primeiro. Em outras palavras, o texto em uma página da Web é “marcado” com esses códigos para instruir o navegador da Web sobre como exibir o texto. Essas tags de marcação são as próprias tags HTML.

Atualmente, existem 142 tags HTML disponíveis que permitem a criação de vários elementos. Embora alguns novos computadores não suportem mais alguns desses, é importante estar ciente da existência deles.

Como funciona a tag HTML?

Os elementos HTML são tipicamente divididos em três partes. Esta combinação dos três cria um elemento HTML.

  • Tag de abertura: Usado para indicar onde um item começa. É envolto em suportes de abertura e fechamento. Por exemplo, você pode usar a tag start p para criar um parágrafo.
  • Conteúdo: O conteúdo é o resultado que o público vê.
  • Tag de fechamento: É o mesmo que a tag de abertura, mas com uma barra na frente do nome do elemento. Ou seja, p para terminar um parágrafo.

Quais são as tags basicas ?

  • body para o conteúdo
  • para informação sobre o documento
  • a para links
  • H1…H6 para títulos dentro do conteúdo
  • br para quebras de linha
  • strong para usar texto em negrito
  • img para adicionar imagens ao documento
  • ol para listas ordenadas, ul para listas desordenadas, li para elementos dentro da lista
  • p para parágrafos

body

Indica a parte do corpo do conteúdo de um documento HTML. É uma tag essencial para qualquer documento, pois indica onde o conteúdo visível do documento começa

head

É a parte superior do documento HTML e onde podemos indicar os metadados: título do documento, estilo, javaScript, CSS…

a

É uma tag que nos ajuda a criar um link para uma página da Web. O principal atributo dessa tag HTML é href, onde colocaremos o link ao qual queremos nos conectar. Outro atributo amplamente utilizado é o target, que serve para indicar se o link será aberto em uma nova janela ou nele.

strong

Se você tem um monte de texto, é importante ser capaz de dar ênfase a uma determinada parte com trechos em negrito, com o rótulo strong podemos fazê-lo.

br

Com esta tag HTML podemos dizer ao navegador que uma quebra de linha está chegando. Serve para tornar o texto mais legível.

h1 h2 h3 h4 h5 h6

Existem diferentes níveis de títulos, de 1 a 6. As tags < H + número> nos permitem indicar a importância do título e estruturar o conteúdo, desta forma ajudamos os bots a entender a importância do conteúdo.

img

Usamos a tag IMG para exibir imagens dentro do conteúdo. Você precisa do atributo SRC para funcionar, pois é aí que vamos indicar de onde você precisa exibir a imagem.

ol ul li

As tags OL e LI são usadas para criar listas, OL para listas ordenadas e UL para listas fora de ordem. Dentro das listas, os itens são identificados com a tag LI.

P

Rótulo que nos serve para agrupar texto dentro de um parágrafo. O objetivo é ser capaz de facilitar a leitura e organização do conteúdo.
